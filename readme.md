# Explode Chess

爆炸棋，支持局域网联机（暂不支持本机同程序直连，可选择开启多个 exe 并以 IP `127.0.0.1` 连入）。

多方轮流下子，某格子数达到本格上限则爆向四周，子数为 0 者负。

## How To Use

对于服务端，在最开始的界面输入 `2` 开启服务器，填写相应地图参数即可进入等待；

对于客户端，在开始界面输入 `1`，输入相应参数及服务端 IP 即可连接。
